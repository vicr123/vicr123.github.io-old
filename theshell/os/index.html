theShell OS

Down at its core, it has theShell's "up-for-anything" attitude.

And the preliminary release is here! Grab the ISO!

For the simple

theShell OS is based on Arch Linux. This allows you to install any software from the Arch Linux repositories, as well as any software from the Arch User Repository, to give you a large software selection.

For the desktop

theShell OS comes with theShell as the default desktop environment. And since theShell OS is created by the same person that made theShell, the desktop is integrated with all the default applications of the OS, giving you the best theShell experience.
